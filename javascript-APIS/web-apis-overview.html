Web API's

  • A Web API is a developers dream
  • It can extend the functionality of the browser
  • It can greatly simplify complex functions
  • It can provide easy syntax to complex code
  • API stands for Application Programming Interface
  • A web API is an application programming interface for the web
  • A browser API can extend the functionality of a web browser
  • A server API can extend the functionality of a web server
  • All browsers have a set of built in web apis to support complext operations and to help accessing data
  • Third Party APIs are not built in the browsers
  • To use third party APIs you have to download the code from the web
