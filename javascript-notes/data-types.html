Data Types

  • In Javascript there are eight different data types:
    • undefined
    • null
    • boolean
    • string
    • symbol
    • bigInt
    • number
    • object
  • Computers distinguish between numbers and strings which are collections of characters.
  • Computers can perform mathematcial operations on a number but not on a string.
  • Variables allow computers to store and manipulate data in a dynamic fashion.
  • Variables use a label to point to the data rather then using the data itself
  • Any of the eight data types can be stored in a variable
  • Variables are similar to the x and y variables you can use in mathmatics, which means they're a simple name to represent the data we want to refer to
  • Computer variables differ from mathematcial variables in that they can store different values at different times
  • We tell JavaScript to create or declare a variable by putting the keyword var in front of it
